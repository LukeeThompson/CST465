Question: What is the purpose of name mangling?

Question: Explain the difference between redirects that use a 301 HTTP status code and ones that use a 302 HTTP status code
Answer: A 301 status code indicates that the resource you are attempting to access can no longer be accessed at the given address. The client should then update its records with the new address. A 302 status code means the resource is temporarily at a new location. The client redirects to the new location but saves the old location for future use.
